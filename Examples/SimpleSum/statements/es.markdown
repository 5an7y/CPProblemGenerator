# Historia

Santy quiere sumar dos numeros

# Problema

Se te daran dos numeros $a, b$ y deberas imprimir su suma.

# Entrada

Dos numeros enteros $a$ y $b$.

# Salida

Un entero indicando la suma de los dos numeros.

# Ejemplos

||input
2 3
||output
5
||description
$2 + 3 = 5$
||input
1099 1
||output
1100
||end

# Limites

- $0 \leq |a|, |b| \leq 10^{17}$
